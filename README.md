# Personal Finance Dashboard

Простое веб-приложение на Django для учета личных финансов, контроля расходов и управления долгами. 

## Функционал
* **Учет транзакций:** добавление доходов и расходов по категориям (еда, подписки и т.д.).
* **Управление долгами:** отслеживание текущих задолженностей с указанием кредитора.
* **Дашборд:** вывод истории операций и текущих долгов на главную страницу.
* **Панель администратора:** управление всеми записями через встроенную админку Django.

## Стек технологий
* **Backend:** Python, Django
* **Database:** SQLite (по умолчанию)
* **Frontend:** HTML, CSS (Django Templates)
